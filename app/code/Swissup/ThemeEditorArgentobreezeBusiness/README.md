# ArgentoBreeze Business Theme Editor

## Installation

### For clients

Please do not install this module. It will be installed automatically as a dependency.

### For developers

Use this approach if you have access to our private repositories!

```bash
composer require swissup/module-theme-editor-argentobreeze-business --prefer-source
bin/magento module:enable Swissup_ThemeEditorArgentobreezeBusiness
```
