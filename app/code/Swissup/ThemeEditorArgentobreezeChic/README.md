# Theme Editor Argento Breeze Chic

## Installation

### For clients

Please do not install this module. It will be installed automatically as a dependency.

### For developers

Use this approach if you have access to our private repositories!

For Swissuplabs developers only!

```bash
cd <magento_root>

composer config repositories.swissup composer https://docs.swissuplabs.com/packages/
composer require swissup/module-theme-editor-argentobreeze-chic:dev-master --prefer-source
bin/magento setup:upgrade
```
