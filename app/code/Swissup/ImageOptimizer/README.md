# Image Optimizer

### Installation

###### For clients

```bash
cd <magento_root>
composer require swissup/module-image-optimizer
bin/magento module:enable Swissup_ImageOptimizer Swissup_Core
bin/magento setup:upgrade
```

###### For maintainers

```bash
cd <magento_root>
composer require swissup/module-image-optimizer:dev-master --prefer-source
bin/magento module:enable Swissup_ImageOptimizer Swissup_Core
bin/magento setup:upgrade
```
